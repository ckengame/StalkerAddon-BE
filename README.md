# STALFEAR Addon (S.T.A.L.K.E.R. for Minecraft Bedrock)

A project aimed at recreating and adapting core elements of the S.T.A.L.K.E.R. universe within Minecraft Bedrock Edition.

---

## Overview

Can you survive in the heart of the Exclusion Zone?

This add-on adds the atmosphere, equipment, and basic mechanics of the S.T.A.L.K.E.R. world to Minecraft. Suitable for both single-player and co-op play, it can also be used as a basis for creating real-world maps and results.

---

## Features

Currently implemented:

- Custom textures
- Firearms and weapon systems
- Wide variety of outfits
- Artifacts
- Anomalies
- NPCs

---

## Planned / In Development

- Custom blocks
- Mutants
- Full anomaly fields system
